Diwali Sales Analysis

Python Project

  1. Home
  2. Diwali Sales

Introduction

This project involves analyzing Diwali sales data using Python. The aim is to uncover valuable customer insights, optimize marketing strategies, and identify top-performing products and customer segments.

Objective

  • Import and clean raw Diwali sales data using Pandas.
  • Explore customer demographics and purchase behavior.
  • Use visualization libraries (Matplotlib & Seaborn) to reveal sales patterns and business insights.
  • Identify profitable customer segments and product categories.

Methodology

  1. Data Import

    The dataset Diwali Sales Data.csv was imported using pd.read_csv().

  2. Data Cleaning

    Unnecessary columns like Unnamed,status were dropped, and missing values handled with dropna() and fillna().

  3. Data Transformation

    New column: Total Amount = Quantity × Amount. Data types were adjusted using astype().

  4. Exploratory Data Analysis (EDA)

    • Gender-wise and age-wise spending patterns
    • Top states by total sales
    • Most profitable product categories

  5. Exporting Cleaned Data

    Saved cleaned dataset as Cleaned_Diwali_Sales.csv.

For more EDA Download

Results

  1. Female customers aged 26–35 were top spenders.
  2. Maharashtra and Karnataka led in sales volume.
  3. Food and clothing categories were most popular.
  4. Married individuals made most purchases.
  5. Targeted campaigns based on age-gender were most effective.
