Introduction

This project analyzes customer behavior and sales patterns in an eCommerce platform using a combination of SQL and Python. The goal is to derive business insights that help improve customer experience, optimize operations, and guide strategic decisions. The data was extracted from a MySQL database and analyzed using Python libraries.

Objective

  • Connect to a MySQL database and extract relevant eCommerce data.
  • Clean and preprocess the data using Python (Pandas, Numpy).
  • Perform SQL-based analysis for trends, customer geography, and product performance.
  • Visualize key insights using Matplotlib and Seaborn.

Methodology

  1. Data Collection

    Data was extracted from a MySQL eCommerce database using Python's mysql.connector. The connection enabled querying key tables such as customers, orders, order_items, products, and payments.

  2. Data Integration

    Multiple SQL tables were joined in Python using pd.read_sql() to create a comprehensive dataset combining customer details, product purchases, and payment history.

  3. Data Cleaning

    Null Value Handling: Nulls in text fields were replaced with "Not Available" and in numeric fields with 0 or -1. This was done using Pandas fillna().

    Duplicate Detection: Duplicate entries were checked using duplicated() and dropped with drop_duplicates().

  4. Data Transformation

    Timestamp fields were converted to datetime using pd.to_datetime(), and additional columns like year, month, weekday were derived for trend analysis.

  5. Data Export

    The final cleaned dataset was saved using df.to_csv('cleaned_ecommerce.csv') for backup and further analysis.

  6. Data Analysis in SQL

    SQL Queries: Python executed SQL queries via cur.execute() to analyze total orders, top customer states, revenue per product, etc.

  7. Analysis

    Order Trends: Tracked number of orders and revenue per year and month.

    Geographic Trends: Identified top customer cities and states.

    Product Performance: Most frequently bought and high-revenue items were identified.
