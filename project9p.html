U.S. Housing Market Analysis

Python Project

  1. Home
  2. Housing Market

Introduction

This project explores U.S. housing market trends using Python. It aims to analyze time-series data for average home prices, mortgage rates, and new home sales to generate business insights.

Objective

  • Load and clean monthly housing data.
  • Convert and manipulate datetime columns.
  • Visualize price and mortgage trends using Matplotlib and Seaborn.
  • Identify top-selling periods and regional differences.

Methodology

  1. Data Import

    The dataset unified_monthly_data_interpolated.csv was imported using pd.read_csv().

  2. Data Cleaning

    Date columns were converted using pd.to_datetime(). Missing and irrelevant values were addressed using dropna() and basic Pandas functions.

  3. Exploratory Data Analysis (EDA)

    • Filtered top records by Average Sales Price
    • Plotted line charts for housing prices and mortgage rates
    • Grouped data by year and region for trend analysis

  4. Exporting

    Cleaned datasets were saved for future reference using to_csv().

Results

  1. Housing prices showed steady growth, especially post-2020.
  2. Mortgage rates spiked in 2022, impacting affordability.
  3. Summer months consistently showed higher home sales.
  4. Eastern and Western U.S. showed regional variations in pricing.
